# Ntando Nkuna Portfolio

## Overview
Streamlit-powered data analytics portfolio showcasing projects, skills, and tech stack.

## Features
- Interactive UI
- Dynamic tech stack display
- Project highlights
- Feedback form

## Installation
pip install -r requirements.txt
streamlit run app/main.py

## Deployment
Deploy via Streamlit Cloud

## Tech Stack
Python, Streamlit, Plotly, SQL, DuckDB
